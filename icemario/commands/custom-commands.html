Custom Commands Manager

Create custom commands to have Ice Mario respond with something! Some example use cases are to get your team's tag or to get someone's twitch link when they're streaming.

Custom commands are enabled by default, but you can disable them through your server's config settings. You can also restrict it to only allow moderators to update them.

There is a limit of 25 commands per server. If you really need to increase the limit, please contact me for assistance.

!customcommandadd command, value
Alias: !cca

command: The name of the command to create
 value: What the command should return

 Add or update a custom command.

There should be a comma separating command and value, and both of them can be more than one word. If you would like Ice Mario to send back an image instead of text, provide the image link to value. All custom commands will use "!" as the prefix. Here is an example of adding the command !invite:

!customcommandremove command
Alias: !ccr

command: The name of the command to create

 Remove a custom command.

For example, to remove the command !invite from above, use !ccr invite

!customcommandlist
Alias: !ccl

List all the custom commands set in a server.
