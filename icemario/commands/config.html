Server Configuration

These are all the server config options currently available, as well as commands to set them. Note: the following commands will only work if you have MANAGE_MESSAGES permission.

im!config

List all the current config options for the server.

im!config key value

Set the config option key with the value value. A list of keys can be found below. For example, to enable the !choose command, use im!config enable-choose true

im!resetconfig

Reset all the server config options to the default values. The default values are shown below.

The following is a list of all the config options as well as their default values:

Clan War Scheduler

  • enable-schedule (Default: true): Enable or disable all scheduler commands and reactions.
  • ping-one-hour-before (Default: true): If set to true, members that have reacted to the schedule will be pinged 1 hour before the scheduled event. If set to false, there will not be such a reminder.
  • ping-15-minutes-before (Default: false): If set to true, members that have reacted to the schedule will be pinged 15 minutes before the scheduled event. If set to false, there will not be such a reminder.
  • ping-when-starting (Default: true): If set to true, members that have reacted to the schedule will be pinged when it is time for the scheduled event. If set to false, there will not be such a reminder.
  • delete-schedule-after-one-hour (Default: false): If set to true, schedules will automatically be deleted 1 hour after the scheduled event. If set to false, the schedule will not be deleted.
  • delete-schedule-after-minutes (Default: false): If set to true, schedules will automatically be deleted delete-schedule-minutes after the scheduled event. Specify the minutes with the config option below.
  • delete-schedule-minutes (Default: 60): The number of minutes for delete-schedule-after-minutes above. This needs to be between 1 and 60.
  • moderators-only-for-scheduler (Default: false): If set to true, only members with MANAGE_MESSAGES permission can create and edit schedule. If set to false, anyone in the server can do those things.
  • gather-size (Default: 4): The number of participants that need to join in order for the schedule to be considered full (the schedule color will turn green).
  • display-username (Default: false): If set to true, the user's username will be shown next to their @ping on each schedule.
  • show-dropped-schedule (Default: false): If set to true, the text "(Dropped)" will be displayed next to people on the can't list who previously reacted with either can or sub.

Clan War Score Tracker

  • enable-score-tracker (Default: true): Enable or disable all score tracker commands.
  • default-team-name (Default: Team1): Sets the default team name for your server. You can use this to set your team tag so that you don't have to set it everytime you use !startwar or !setteams.
  • dynamic-color (Default: true): If enabled, the left side of Ice Mario's message will be green if your team is winning and red if your team is losing. The color will be darker the greater the total score difference. If disabled, a sky-blue color will be used.
  • show-score-graph (Default: true): Whether or not to display a graph of your team's total score difference for the war.

Table Maker

  • enable-table-maker (Default: true): Enable or disable all table maker commands. Note that this does not have any effect for logo commands such as !addlogo or !getlogo.
  • enable-logo-colors (Default: true): Enable or disable displaying colors when using the !tablel command. If set to false, the text will be white by default, but you can still set the color for a team by specifying it to the right of the team name (like DY #FF0000).

Lounge

  • enable-lounge-stats (Default: true): Enable or disable all MKTC lounge commands such as !mmr and !stats.

Mogi

  • enable-mogi-gatherer (Default: false): Enable or disable all mogi commands.
  • room-size (Default: 8): The number of people needed to start a mogi.
  • max-subs (Default: 2): The number of people that can !c once a mogi has started.
  • inactive-minutes-until-drop (Default: 25): The number of inactive minutes before someone is dropped from the mogi.
  • minutes-to-ping-before-drop (Default: 5): The number of minutes before inactive-minutes-until-drop to ping someone to have them type something in the chat.
  • max-voting-minutes (Default: 2): The number of minutes to wait for people to vote on the mogi format after it has started. If the votes did not get 4 votes, then this will close the mogi voting once time is up.
  • should-ping-here (Default: true): Whether or not to ping @here.
  • only-moderator-start (Default: false): Whether or not only members with MANAGE_MESSAGES can start a mogi.
  • only-moderator-end (Default: false): Whether or not only members with MANAGE_MESSAGES can end a mogi.
  • only-moderator-remove (Default: false): Whether or not only members with MANAGE_MESSAGES can remove someone from a mogi.
  • enable-voting (Default: false): Enable or disable all voting one a mogi fills.
  • enable-voting-ffa (Default: true): Enable or disable the option to vote for ffa.
  • enable-voting-2v2 (Default: true): Enable or disable the option to vote for 2v2.
  • enable-voting-4v4 (Default: true): Enable or disable the option to vote for 4v4.
  • randomize-teams-2v2 (Default: true): Whether or not to randomize 2v2 teams when voting is complete.
  • randomize-teams-4v4 (Default: true): Whether or not to randomize 4v4 teams when voting is complete.

Custom Commands

  • enable-custom-commands (Default: true): Enable or disable custom commands in your server.
  • allow-anyone-to-update-cc (Default: true): If set to true, anyone in the server can add, update, or delete custom commands. If set to false, only people with MANAGE_MESSAGES permission can do so.

Miscellaneous

  • enable-choose (Default: false): Enable or disable the !choose command.
  • enable-randomize-teams (Default: false): Enable or disable the !teams command to randomize teams for a mogi.
  • enable-say (Default: true): Enable or disable the !im command to make Ice Mario talk.
