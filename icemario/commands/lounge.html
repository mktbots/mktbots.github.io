MKT Lounge

Commands to get MKTC Lounge stats. If needed, you can disable these commands through your server's config settings.

!mmr nameList or
!mmr teamNum nameList

teamNum (optional): The number of teams to split the names into. Teams are based on the order of nameList
nameList (optional): 1 or more names separated by commas.

Get the mmr of the command user or a list of names. An average mmr will also be calculated if there are two or more names.

Examples:

  • !mmr: Get your mmr. Make sure your server nickname matches one from the leaderboard.
  • !mmr player1: Get the mmr of player1.
  • !mmr player1, player2, player3, player4: Get the mmr values of all four players.
  • !mmr 4 player1, player2, player3, player4, player5, player6, player7, player8: Separate into 4 teams and get the average mmr for each team

!stats name

name (optional): Someone's name or their member ping (@someone).

Get the last 10 stats.

Examples:

  • !stats: Get your last 10 stats. Make sure your server nickname matches one from the leaderboard.
  • !stats bob: Get the last 10 stats for the player named bob.

!stats# name

#: The season number.
name (optional): Someone's name or their member ping (@someone).

Get the history stats for a particular season.

Examples:

  • !stats7: Get your season 7 stats.
  • !stats10 bob: Get bob's season 10 stats.

!statsall name

name (optional): Someone's name or their member ping (@someone).

Get the history stats from all seasons starting with season 1.

!last# name

# (optional): Specify the last number of events. If left out, this will get the whole season.
name (optional): Someone's name or their member ping (@someone).

Get the stats of the last # events.

Examples:

  • !last15: Get your stats for the last 15 events.
  • !last10 bob: Get bob's stats for the last 10 events.

!top# rank

# (optional): The season number. If left out, this will get the current season.
rank (optional): Returns the leaderboard page containing rank.

Get the lounge leaderboard. Buttons are included to switch between pages.

Examples:

  • !top: Get the top 10 players for the current season.
  • !top8: Get the top 10 players for season 8.
  • !top 20: Jump to the page containing the player with rank 20.
  • !top11 30: Jump to the page containing the player with rank 20 for season 11.

!predict tableData

Predict the mmr change for 8 players. There needs to be 8 players as well as 1, 2, or 4 teams.

Player points are not used in the calculation, they are only used to find each team's rank. If you want to predict the mmr gain before a migi starts, you can enter a small value like 1 to predict your gain if you get first place.

Note: this command is to predict mmr changes, so the actual change may be off by a few points depending on how many tables still need to be updated.

!scoreboard or
!scoreboard # player1, player2, ..., player8

Generate the scoreboard for a mogi. Note: This command is only available in the MKTC lounge server and in servers where enable-mogi-gatherer is set to true.

!mmrlu

Get the mmr of the current mogi lu. This command is only available in the MKTC Lounge server.
