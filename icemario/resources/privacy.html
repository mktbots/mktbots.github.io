Privacy Policy


References

In this document, "we" or "us" refers to the bot Ice Mario as well as developers.

"Personal data" includes discord IDs and other information accessible to every other discord user as well as personal information you pass on via a command usage.


Data collection

We do not collect any kind of personal data that you didn't authorize. However, information needed to operate will be stored. The following list, which may not be exhaustive, will show what data is collected:

  • Discord server ID
  • Discord channel ID
  • Discord message ID
  • Discord user ID
  • Images (in regards to custom logos and flags)

Message content outside of a command and other personal user-generated content are not stored in any way. This means we don't save anything you send on Discord if you are not interacting with the bot.

We reserve the right to collect data whenever a command is used, and that data may be stored for analytical purposes.


Distribution of data

None of your data is distributed in any way, and it is only used in internal processes to improve user experience and services.


Deletion of data

All user-related data can be deleted on request. Go to our contact page to see how you can contact us.

MKT Bots and all Discord bots featured are not affiliated with Nintendo.
