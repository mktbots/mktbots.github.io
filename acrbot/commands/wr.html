World Record Commands

Find out what the current action count world record is for any track.

/wr

track-name: The name of the track. Click here for an explanation on how track names are resolved.

Get the action count world record for a track.

/wrsubmit

track-name: The name of the track.
ac: The action count.
player-name: The name of the player.
tour-name (optional): The name of the tour. The bot will automatically remove the word "tour" as well as parentheses, or if tour-name is left out it will automatically use the name of the current tour.
video-link (optional): An optional video link.

Submit a new action count world record for a track. Before submitting, make sure to read the rules found by using the /wrrules command.

If you notice any issues with a submission after it has been approved, such as forgetting to include a video link, you may resubmit to correct it.

Note: /wrsubmit is only available on the ACR Bot server.

/wrrules

Get the rules required to submit with /wrsubmit.

As of the Night Tour, these are the rules:

  1. The race shall be in 150cc.
  2. The race shall be finished in 1st.
  3. Any boosted skills (i.e. those not available on the course as shown by /trackinfo) are not eligible.
  4. Records made with pumpkins are not eligible. However, Starting with New Year's 2023 Tour, events/festival obstacles (Christmas Trees / Katdomatsus) are eligible if there is video proof.
  5. Starting with the Night Tour, tied records are allowed.
  6. Battle tracks are not eligible.
  7. Player names cannot include the following characters: :().

/wrtour

tour-name The name of the tour.

 Get the list of world records that were set during the specified tour. Records set before or after tour-name are not included.

/wrcurrenttour

Get the action count world records for each track of the current tour. This is different than /wrtour in that /wrcurrenttour shows the current WR for every track of the current tour regardless of when the WR was set.

/wrlist

player-name: The name of the player. Use * at the beginning or end to do a wildcard search.

List the action count world records for a player.
