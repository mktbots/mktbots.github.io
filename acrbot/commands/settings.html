User Settings

/settings list

List your current settings. If you have not used /settings set before, there will be nothing to list.

/settings set

Set a setting option. A list of available settings are found below:

  • trackinfo-middle-shelf-driver: Choose either yes or no. If yes, using /trackinfo will always show middle shelf coinbox drivers if they exist for the track.
  • show-wr-with-trackinfo: Choose either yes or no. If yes, using /trackinfo will also show the current AC world record card.
  • score-nickname: Set your nickname for all /score commands. This will also update all your previous score submissions to reflect the name change. If you no longer want to use a nickname, enter None and your Discord username will be used instead.
