Helpful Tips

Track Name Parser

Commands requiring the argument track-name now have autocomplete enabled. Essentially, while you are typing, a list of track names will appear. Note however, autocomplete only shows suggestions and is not required.

Overall, these are the general guidelines:

  • The track name is case insensitive.
  • Entering the full track name (including the console like snes or gba) is the most accurate method. Ex: 3ds mario circuit.
  • In general, you can leave out the console name. However, some tracks that share the same abbreviation, like bc3, requires it.
  • You may use the track name's acronym or abbreviation. Ex: nbc instead of neo bowser city.
  • For normal track variants, do not include n at the end of the track's name. Ex: use bp instead of bpn to get the normal variant of Baby Park.
  • You can use rt instead of r/t.
Here are some examples that will all resolve to SNES Choco Island 1R:
  • snes choco island 1r
  • choco island 1r
  • snes ci 1r
  • snes ci1r
  • snesci1r
  • ci1r
In some cases, the track acronym is shared between two or more tracks. For example, cm can apply to Choco Mountain and Coconut Mall. In such cases, you should either include the console name or use one of the abbreviations below.

For commands like /trackinfo or /wr, there are additional ways to specify track-name:
  • Add the word "all" to get all track variants at once. For example, nym all or nymall will get the N, R, T, and R/T variants all at once.
  • Use week 1 or w1 to get the three tracks for the first week of ranked cup.
  • Similarly, use week 2 or w2 for the second week.


Tour Name Parser

Similar to the trackname parser, commands requiring the argument tour-name now have autocomplete enabled, and this will be the most accurate way to specify the tour name.

Overall, these are the general guidelines:

  • The tour name is case insensitive.
  • Adding the word tour to the end is optional as the bot will automatically add it if it's missing.
  • For repeated tour names, adding parentheses around the year is optional.
Here are some examples that will resolve to Night Tour:
  • night tour
  • night
Here are some examples that will resolve to Mario vs. Luigi Tour (2023):
  • mario vs. luigi tour (2023)
  • mario vs. luigi (2023)
  • mario vs luigi 2023
In the event you enter a tour name that cannot be resolved, like if you used mario vs luigi without specifying the year, the bot will send you a list of similar tour names.
